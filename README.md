## Description: 
This is a project to set up a team of employees using class constructors. Each employee is a manager, engineer, or intern and data is requested from the user for each member of the team via the terminal. This project uses the inquirer npm package and was constructed using TDD techniques. The test package utilized was jest. There is a link to a shared video via google drive at the top of this readme illustrating the functionality of the app.

## Installation: 
This project is installed by cloning the repo to your local workspace and utilizing the npm i command to install the associated package (inquirer). Once the package is installed, you can run the app in your terminal. Once you are finished building your team, an html file is saved and you can open it in your browser, formatted for easy use.

## Usage: 
This project is used in order to quickly build and access some contact information for employees as entered by the user.
